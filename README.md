# DocSentinel v2

AI-powered document intelligence platform — by PhRedSec™ Private Limited

## What it does

DocSentinel is an all-in-one AI document platform for Indian businesses. It reads, extracts, validates, and secures every document your business touches — automatically.

### Four AI modules
- **Smart Vault** — upload any document, ask questions in plain English, AES-256-GCM encryption
- **Invoice & GST Intelligence** — extract vendor, amount, GSTIN, HSN codes, validate against government API
- **Contract Intelligence** — flag risky clauses, track deadlines, email alerts before expiry
- **HR Document Manager** — offer letters, payslips, PAN, Aadhaar, role-based access

## Tech stack

- **Backend** — Python 3.11 + FastAPI + PostgreSQL
- **Frontend** — React + Vite
- **AI** — Groq API (LLaMA 3.3 70B + LLaMA Vision)
- **Storage** — Cloudflare R2
- **Hosting** — Railway (backend) + Vercel (frontend)
- **Encryption** — AES-256-GCM
